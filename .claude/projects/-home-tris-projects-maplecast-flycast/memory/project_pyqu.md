putyourquarterup.com is the planned product layer on top of Maplecast. Decentralized arcade: community members host MVC2 on their own PCs, players connect via browser, every match costs a $QUARTER (Solana SPL token, ~$0.25).

**Why:** Eliminates server costs — hosts donate GPU compute in exchange for earning $QUARTER per match (20% host cut, 10% PYQU, 70% winner). A $60 used Dell pays for itself in ~3 days.

**How to apply:** All streaming/input tech decisions should consider that the end deployment is community-hosted PCs, not data centers. Host client needs to be one-click installable. The full plan lives in docs/PUTYOURQUARTERUP.md.

Key features: spectator betting, on-chain tournament brackets, regional turf wars, physical arcade cabinets in bodegas, NOBD identity integration, cross-game expansion beyond MVC2.
